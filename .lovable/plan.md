# Friday 5-Min Debrief — Final Plan

## The Core Loop

Every Friday at **9am local time**, every active user gets a Friday Debrief invite (email at launch, SMS later). They respond via **any channel** — email reply, voice with Jericho, web chat, or (later) SMS voice memo. Within 1 hour, Jericho replies with a short synthesis of what he heard + 1 sharp next-step nudge. Themes get extracted, scored across 5 categories, and woven back into Daily Briefs, Growth Plan, and Playbook recommendations. Managers see themes/trends only — never raw words.

## The 4 Questions

Sent as a warm, conversational prompt — not a form:

1. **Wins** — What won this week? (small or big, count it)
2. **Stuck** — Where did you get stuck or frustrated?
3. **Focus** — What are you actively working on right now?
4. **Need** — What do you need this week — from me, your team, a customer, or yourself?

Voice/chat = conversational ("alright, talk to me — what won this week?"). Email = one short paragraph + the 4 prompts inline. Replies of any length count — even 2 sentences.

## The 5 Tracked Categories (the Reflection Radar)

Jericho extracts and scores 0–10 from each debrief:

- **Wins** — frequency + weight of wins called out
- **Stuck** — inverted; high score = unstuck and flowing
- **Focus** — clarity vs scatter on what they're working on
- **Asks** — whether they're naming what they need + whether prior asks got resolved
- **Vibe** — overall sentiment / energy

Users never rate themselves. The system extracts.

## The Reflection Radar (Hero Visual)

On the **My Growth Plan** page — new "Weekly Reflections" section:

### 1. The Radar
5-axis radar chart (Recharts `RadarChart`). Two overlapping shapes: **this week** (filled, brand color) vs **4-week average** (outline). Instantly shows where they're above/below their own baseline.

### 2. The Trend Strip
Below the radar: 5 sparklines (one per category) showing last 12 weeks + delta vs last week.

### 3. The Story
1-paragraph auto-narrative below the visuals, e.g.:
> *"Wins are up 2 points from last month — third week of consistent momentum on Prairie Vista. Stuck is climbing though, mostly around pricing pushback. You've asked for help on it twice and haven't gotten an answer — want me to draft something?"*

Generated by Gemini Flash from the last 4 debriefs. Refreshes weekly.

### 4. Streak + History
Streak badge ("6 weeks 🔥") top-right. Below the visuals, last 4 debriefs as expandable cards: date, channel icon (✉️ 🎙️ 💬), and the 4 fields.

Week 1 with one debrief = solid radar, no comparison overlay yet — still satisfying.

## Where the Data Shows Up

1. **My Growth Plan page** — Reflection Radar + trend + story + history (above).
2. **Monday Daily Brief** — opens with last debrief callback + tiny inline radar (5 numbers + delta vs last week, HTML table).
3. **Playbook regeneration** — debrief themes feed recommendations (3 weeks of climbing Stuck on objection handling → suggest SPIN module).
4. **Content / AI nudges** — recurring frustration triggers proactive resource in next Daily Brief.
5. **Manager Dashboard** — team-aggregate radar + per-rep mini-radar. Themes only, never raw words. Respects per-user opt-out.

## Engagement Mechanics

- **Streak counter** — Growth Plan, Daily Brief, banner.
- **Callback** — every Friday email opens with *"Last Friday you said: '...'"* — proof they're being heard.
- **1-hour Jericho reply on every debrief** — short synthesis + 1 next-step nudge. Sent async via existing email queue.
- **Soft re-engagement** — miss a week → Saturday 10am local: *"no debrief landed yesterday — 2 quick min now, or skip this week?"*. Two misses → Jericho asks if rhythm is wrong.
- **Time customization** — default Fri 9am local; user picks Thu PM / Fri AM / Fri PM / Sun PM in Settings.
- **Anytime debrief** — never blocked by schedule.

## Rollout

**Default ON for all active users.** One-click pause in Settings + unsubscribe link in every Friday email.

- Week 1: ship to internal team + 5-user beta
- Week 2: roll out to all active users with in-app announcement + Daily Brief mention

## Technical Plan

### New tables

- **`friday_debriefs`** — one row per submitted debrief
  - `id`, `profile_id`, `week_of` (Monday of that week), `submitted_at`, `channel` (email/voice/web/sms), `raw_response` (text), `transcript` (voice), `wins_text`, `stuck_text`, `focus_text`, `need_text`, `extracted_themes` (jsonb), `category_scores` (jsonb: `{wins, stuck, focus, asks, vibe}` 0-10), `narrative_summary` (text, the 1-paragraph story), `audio_url` (nullable), `processed_at`, `jericho_reply_sent_at`
  - RLS: user reads/writes own; managers via security-definer RPC (aggregates only)
  - Unique on (profile_id, week_of)

- **`friday_debrief_invites`** — `id`, `profile_id`, `week_of`, `invited_at`, `channel_invited`, `responded_at`, `reminder_sent_at`, `skipped`

- **`friday_debrief_streaks`** — `profile_id` (PK), `current_streak`, `longest_streak`, `last_debrief_week`, `total_debriefs`

- Extend `user_active_context`: `friday_debrief_enabled` (bool default true), `friday_debrief_day_time` (text default `'fri_9am'`), `friday_debrief_share_with_manager` (bool default true)

### Edge functions

1. **`send-friday-debrief-invite`** (cron, hourly Fri/Sat) — finds users whose local time matches their slot, not yet invited this week. Sends `friday-debrief-invite` template via `send-transactional-email` with last-week callback.

2. **`process-friday-debrief`** — single intake handler called from email-reply, voice end-of-session, web chat, future SMS.
   - Uses Gemini Flash to parse the 4 fields, extract themes, score 5 categories, write narrative
   - Inserts `friday_debriefs`, updates streaks
   - Enqueues 1-hour Jericho reply via `send-transactional-email` (template: `friday-debrief-jericho-reply`)

3. **`friday-debrief-reminder`** (cron, hourly Sat) — sends soft reminder to non-responders.

4. **`get-debrief-context-for-brief`** — utility for Daily Brief generator. Returns last debrief + 4-week trend block.

5. **`generate-debrief-narrative`** — utility called when rendering Growth Plan radar; generates/refreshes the 1-paragraph story. Cached on `friday_debriefs.narrative_summary`.

### Routing changes to existing functions

- **`process-email-reply` / `sales-intent.ts`** — add `friday_debrief` intent. Detection: subject contains "Friday Debrief" OR `In-Reply-To` header references a debrief invite message-id. Routes to `process-friday-debrief`, short-circuits other handlers.
- **Daily Brief generator** — call `get-debrief-context-for-brief` on Mondays; inject `last_friday_debrief` block + tiny radar HTML table.
- **Playbook engine** — pull last 4 debriefs' themes + scores into context.

### New email templates (in `_shared/transactional-email-templates/`)

- `friday-debrief-invite.tsx` — warm intro + last-week callback + 4 questions inline + "Just hit reply with whatever lands" + buttons for voice/chat + streak badge if ≥2 weeks. Brand styling per email standards memory (deep navy + gold).
- `friday-debrief-reminder.tsx` — softer, shorter.
- `friday-debrief-jericho-reply.tsx` — "Got it — here's what I heard:" synthesis + 1 next-step nudge.

### UI changes

- **`MyGrowthPlan` page** — new "Weekly Reflections" section with Radar + Trend Strip + Story + Streak + History (above).
- **Dashboard banner** — Fri/Sat morning if pending: "Your Friday Debrief is waiting — 5 min, your call on how" with 3 buttons: Reply by email / Voice with Jericho / Quick chat.
- **Settings page** — new "Friday Debrief" section: enable toggle, time picker, share-with-manager toggle.
- **`JerichoVoiceChat` / `JerichoVoiceChatOpenAI`** — Friday Debrief mode (`?mode=friday_debrief`): walks through 4 questions conversationally, posts transcript to `process-friday-debrief` on session end.
- **`JerichoChat`** — same mode param; guided 4-question flow with skip-per-question (Flexibility Principle).
- **`ManagerDashboard`** — new "Team Reflections" panel: aggregate radar + per-rep mini-radars + theme chips. Behind security-definer RPC.

### Cron jobs

```sql
select cron.schedule('friday-debrief-invites', '0 * * * 5,6', $$ ... send-friday-debrief-invite ... $$);
select cron.schedule('friday-debrief-reminders', '0 * * * 6', $$ ... friday-debrief-reminder ... $$);
```

Add core line: "Monday Daily Brief always opens with last Friday's debrief callback if available."
